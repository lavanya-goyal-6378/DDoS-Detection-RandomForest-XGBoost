# Network Traffic Classifier
 
Supervised ML model to classify network flows into traffic types
(BENIGN, DDoS) using the CICIDS 2017 dataset.
 
## Results
| Model         | Accuracy | F1 (macro) |
|---------------|----------|------------|
| Random Forest | 98.2%    | 0.951      |
| XGBoost       | 99.1%    | 0.971      |
 
## Tech Stack
Python, scikit-learn, XGBoost, pandas, SMOTE
 
## Dataset
CICIDS 2017 — Canadian Institute for Cybersecurity
